\section{Parsing Assumptions and Coverage}
\label{app:parsing-coverage}
\setcounter{table}{0}
\setcounter{figure}{0}
\begin{itemize}
    \item Format assumed: Mon DD HH:MM:SS host process[pid]: message; no year in timestamps.
    \item Asset assumption: \texttt{backup01} stores production backup data and supports production recovery.
    \item Account-role assumption: \texttt{backup} is a non-interactive backup service identity; named accounts perform human administration.
    \item Username handling covers both \texttt{Failed password for USER} and \texttt{Failed password for invalid user USER}.
    \item Non-standard connection-close and reverse-mapping warning lines are classified or counted as unmatched, not silently dropped.
    \item Unmatched-line count: 0.
\end{itemize}
